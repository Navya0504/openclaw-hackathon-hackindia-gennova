# AI Fake News Detector for WhatsApp Links

This project verifies suspicious WhatsApp messages and links.

Features:
- Detects suspicious URLs
- Checks HTTPS security
- Scans webpage content for phishing keywords
- Provides a security verdict (REAL / FAKE)

Tech Stack:
Python
Flask
BeautifulSoup
Requests
HTML
CSS
